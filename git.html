Trucos GIT necesarios

Ver modificaciones de codigo


	// ver los cambios del codigo antes de hacer commit
	git diff



	// ver los cambios del codigo de un commit
	git diff numero-de-commit

Conocer cuales fueron los ultimos archivos que se modificaron


	// Conocer cuales fueron los ultimos archivos que se modificaron

	$ git log --oneline -2 --name-only
	c4a087a (HEAD -> master) commit 4
	ruta git.txt
	5056785 commit 3 - agrego div cuadrado
	hola_mundo.html

Donde -2 indica los 2 ultimos commit y --name-only indica el nombre del archivo modificado
