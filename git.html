Trucos GIT necesarios

Ver modificaciones de codigo


	// ver los cambios del codigo antes de hacer commit
	git diff

Supongamos que realice cambios en varios archivos pero quiero saber las modificaciones de codigo de solo uno


  	// ver los cambios del codigo de un commit
  	git diff numero-de-commit



	// ver los cambios del codigo de un archivo
	 git diff c4a087a 'ruta git.txt'



	// Saber que nombres de archivos se editaron
	 git log --name-only 



	// Saber que nombres de archivos se editaron en commit especifico
	 git log --name-only c4a087a

consultar solo los ultimos commit

donde -3 indica la cantidad de lineas a mostrar


	// consultar solo los ultimos commit
	git log --oneline -3
